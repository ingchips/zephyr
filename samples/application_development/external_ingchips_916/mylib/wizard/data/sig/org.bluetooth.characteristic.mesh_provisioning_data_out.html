Mesh Provisioning Data Out

org.bluetooth.characteristic.mesh_provisioning_data_out

Abstract
The Mesh Provisioning Data Out characteristic can be notified to send a Proxy PDU message containing Provisioning PDU from a Provisioning Server to a Provisioning Client.
Value
NameRequirementFormatNote
Provisioning PDU

The characteristic value is 66 octets long to accommodate the longest known Proxy PDU message containing Provisioning PDU.		struct
