IEEE 11073-20601 Regulatory Certification Data List

org.bluetooth.characteristic.ieee_11073-20601_regulatory_certification_data_list

Abstract
The value of the characteristic is an opaque structure listing various regulatory and/or certification compliance items to which the device claims adherence.
Summary

The content of this characteristic is determined by the Authorizing Organization that provides Certifications. Refer to 11073-20601 [1] or Continua Design Guidelines [2] for more information on the format of this list.
