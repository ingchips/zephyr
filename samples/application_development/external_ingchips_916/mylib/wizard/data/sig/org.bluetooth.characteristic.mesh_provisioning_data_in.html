Mesh Provisioning Data In

org.bluetooth.characteristic.mesh_provisioning_data_in

Abstract
The Mesh Provisioning Data In characteristic can be written to send a Proxy PDU message containing Provisioning PDU to the Provisioning Server.
Value
NameRequirementFormatNote
Provisioning PDU

The characteristic value is 66 octets long to accommodate the longest known Proxy PDU containing Provisioning PDU.		struct
