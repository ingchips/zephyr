Internet Protocol Support Service

org.bluetooth.service.internet_protocol_support

Abstract
When in a GAP Discoverable Mode for an initial connection to a Router, the Node will include the IP Support Service UUID 0x1820 in the Service UUIDs AD type field of the advertising data. This enhances the user experience as a Node may be identified by the Router before initiating a connection.
