Mesh Proxy Data In

org.bluetooth.characteristic.mesh_proxy_data_in

Abstract
The Mesh Proxy Data In characteristic is used by the client to send Proxy PDUs to the server.
Value
NameRequirementFormatNote
Proxy PDU

The Mesh Proxy Data In characteristic shall support Proxy PDU messages containing Network PDUs, mesh beacons, and proxy configuration messages and shall not support other Proxy PDU type messages.		struct
