Mesh Proxy Service

org.bluetooth.service.mesh_proxy

Abstract
The Mesh Proxy Service is used to enable a server to send and receive Proxy PDUs with a client.

Mesh Proxy Data Out
org.bluetooth.characteristic.mesh_proxy_data_out

The Mesh Proxy Data Out characteristic is used by the server to send Proxy PDUs to the client. The Mesh Proxy Data Out characteristic shall support Proxy PDU messages containing Network PDUs, mesh beacon, and proxy configuration messages and shall not support other Proxy PDU type messages.
